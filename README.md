# 2026년 병오년 설 선물세트 웹 브로슈어

## 📦 프로젝트 개요
경상남도장애인생산품판매시설의 2026년 설 명절 선물세트 전체 카탈로그 웹 브로슈어입니다.
총 68개 제품(경남도내 12개, 경남도외 56개)을 온라인에서 쉽게 열람할 수 있도록 제작되었습니다.

레퍼런스 웹사이트 스타일(e-catalog)로 재구성된 모던하고 깔끔한 디자인의 웹 카탈로그입니다.

## 🐴 2026 병오년 (丙午年)
말의 해를 맞이하여 준비한 특별 선물세트 카탈로그입니다.

## 📁 파일 구조
```
web_brochure_2026/

└── README.md           # 이 파일
```

## 🚀 사용 방법

### 1. 로컬에서 열기
- `index.html` 파일을 더블클릭하거나
- 웹 브라우저에 드래그 앤 드롭
- 로컬 웹 서버 실행 (권장):
  ```bash
  # Python 3
  python -m http.server 8000
  
  # Node.js (http-server)
  npx http-server
  ```

### 2. Vercel 배포 (권장)
```bash
# Vercel CLI 설치
npm i -g vercel

# 프로젝트 디렉토리에서 배포
vercel

# 또는 GitHub에 푸시 후 Vercel 대시보드에서 연결
```

**장점:**
- 무료 호스팅
- 자동 CDN (이미지 빠른 로딩)
- 자동 HTTPS
- Git 연동 시 자동 배포

### 3. GitHub Pages 배포
```bash
git init
git add .
git commit -m "2026 병오년 설 선물세트 브로슈어"
git branch -M main
git remote add origin [your-repo-url]
git push -u origin main
```
Settings → Pages → Source: main branch 선택

### 4. 일반 웹 서버에 업로드
전체 폴더를 웹 호스팅 서버에 업로드하면 됩니다.
- `images/` 폴더와 모든 파일이 같은 디렉토리에 있어야 합니다.
- HTML 파일만 업로드해도 모든 기능이 작동합니다.

## ✨ 주요 기능

### 1. 전체화면 이미지 뷰어
- 제품 클릭 시 전체화면 모달로 이미지 확대
- 키보드 화살표 키로 이전/다음 제품 탐색
- 터치 스와이프 지원 (모바일)
- ESC 키로 닫기
- 제품 정보 표시 (이름, 설명, 가격)

### 2. 제품 필터링
- 전체보기
- 경남도내 / 경남도외 구분
- 가격대별 필터 (3만원 이하 / 3~5만원 / 5만원 이상)

### 3. 검색 기능
- 제품명, 카테고리, 가격으로 실시간 검색
- Enter 키 또는 검색 버튼으로 검색

### 4. 반응형 디자인
- 데스크톱, 태블릿, 모바일 모든 기기 지원
- 자동 그리드 레이아웃 조정
- 터치 친화적 인터페이스

### 5. 사용자 경험
- 제품 카드 호버 효과
- 부드러운 스크롤 애니메이션
- 스크롤 기반 fade-in 애니메이션
- 이미지 lazy loading
- 맨 위로 버튼
- 로딩 상태 처리

## 🔧 커스터마이징

### 제품 추가/수정
`products.js` 파일에서 `productsData` 객체를 수정:
```javascript
{
    id: 번호,
    name: '제품명',
    category: '카테고리',
    desc: '설명',
    price: 가격(숫자),
    unit: '(단위)',
    price2: 추가가격(선택),
    unit2: '추가단위(선택)',
    img: 'images/파일명.jpg'
}
```

### 스타일 변경
`style.css` 파일의 `:root` 섹션에서 색상 변수 수정:
```css
:root {
    --primary: #2C3E50;      /* 메인 색상 */
    --accent: #E74C3C;       /* 액센트 색상 */
    --gold: #F39C12;         /* 금색 */
    --bg: #FFFFFF;           /* 배경색 */
    --text: #2C3E50;         /* 텍스트 색상 */
}
```

### 이미지 경로 변경
이미지를 외부 호스팅(Supabase Storage 등)에 업로드한 경우:
1. `config.js` 파일 생성 (필요시)
2. `products.js`의 이미지 경로를 업데이트
